### Opening the console

To open the console and achieve limitless... potential, you just press the backquote "`" key or tilda "~", that'll bring up the console for you were you can type a wide array of commands

The console is meant to accurately emulate the source 1 console as much as possible, in terms of responses to commands, commands that exist and more.

### add_gabes

Add gabes is the first command I added, orginally used to give you free clicks but I have since forgotton about the clicker feature so this command is mostly obsolete

### mute_music [(true/1, false/0]

Stops the music from playing on the website, as it can be annoying after a while... I know.

### set_music [music_url]

leave the first arg blank to unset custom music. 

This changes the music that plays when you load on the page to whatever you want, as long as its an mp3 file because i am file-typist!
